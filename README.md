# React Netflix Clone

## App Features

- Browse Trending Movies: View currently popular movies with details
- Explore TV Series: Discover trending TV shows and series
- Search Functionality: Search across both movies and TV series
- View Movie and Series Details
- Add Favorites 
- Store to Local Storage

## Technical Features

- Movie API integration to get movie and series info
- Local Storage in JS to store 
- React Router to switch between pages
- NPM 
- useEffects and useContext
